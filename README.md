# MERN Task Manager

A full-stack Task Manager application built using the MERN stack.

## Features
- JWT Authentication
- User-specific tasks
- CRUD operations
- Search, filter, sort
- Pagination
- Responsive UI
- Redux state management

## Tech Stack
- Frontend: React, Redux, CSS
- Backend: Node.js, Express
- Database: MongoDB

## Setup

### Backend
```bash
cd server
npm install
npm run dev
